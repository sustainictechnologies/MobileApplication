# JALAD – Smart Water Refill

Flutter mobile app + Node.js/PostgreSQL backend.

---

## Project Structure

```
MobileApplication/
├── jalad_application_1/   ← Flutter mobile app
└── jalad_api_server/      ← Node.js REST API
```

---

## Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [Node.js](https://nodejs.org/) v18+
- [PostgreSQL](https://www.postgresql.org/) running locally

---

## 1. API Server Setup

```bash
cd jalad_api_server
npm install
```

Copy the env file and fill in your values:

```bash
cp .env.example .env
```

Edit `.env`:

```
DATABASE_URL=postgresql://user:password@localhost:5432/jalad_db
JWT_SECRET=<generate with: node -e "console.log(require('crypto').randomBytes(64).toString('hex'))">
PORT=3000
```

Set up the database:

```bash
npm run seed
```

Start the server:

```bash
npm run dev
```

## 2. Flutter App Setup

```bash
cd jalad_application_1
flutter pub get
```

### Finding your local IP

**Windows:** Open Command Prompt → `ipconfig` → look for IPv4 Address  
**Mac/Linux:** Open Terminal → `ifconfig` → look for `inet` under your Wi-Fi interface

### Running on a physical device

Plug in your Android device with USB debugging enabled, then:

```bash
flutter run --dart-define=API_BASE=http://<YOUR_LOCAL_IP>:3000/api
```

**Example:**

```bash
flutter run --dart-define=API_BASE=http://192.168.1.3:3000/api
```

> Your phone and laptop must be on the **same Wi-Fi network**.

### Running on an emulator

```bash
flutter run --dart-define=API_BASE=http://10.0.2.2:3000/api
```

> Android emulator uses `10.0.2.2` to reach the host machine's localhost.

---

## 3. Environment Variables

| Variable | Description |
|---|---|
| `API_BASE` | Full API base URL including `/api` suffix |

The app has a safe default (`http://192.168.1.3:3000/api`) but **always pass your own IP** to avoid connection errors.
